一、理解ECMAScript和JavaScript的关系

  • ECMAScript是一种脚本语言规范，JavaScript是编程语言；
  • JavaScript是ECMAScript语法的一种最为流行的具体实现（除此之外比如 Flash里的ActionScript也是一种实现）；
  • JavaScript是基于ECMAscript规范的脚本语言，并在此基础上实现了自己的封装。
